# 자율주행 데이터 취득 내비게이션

차량 기반 자율주행 데이터 취득 작업을 지원하기 위한 모바일·웹 내비게이션 PWA입니다.

지역과 수집 시나리오를 선택하면 실제 도로 경로, 현재 주행 구간, 예상 시간, GPS 위치, 턴바이턴 안내를 확인할 수 있습니다.

## 기술 구성

| 구분 | 사용 기술 |
| --- | --- |
| 화면 | HTML5, CSS3, JavaScript |
| 지도 | Leaflet 1.9.4 |
| 지도 타일 | OpenStreetMap |
| 도로 경로 | OSRM Routing API |
| GPS | Browser Geolocation API |
| 음성 안내 | Web Speech API |
| 엑셀 생성 | SheetJS |
| 데이터 저장 | Browser localStorage |
| 오프라인 지원 | Service Worker |
| 배포 | GitHub Pages |
| 앱 형태 | Progressive Web App |

---

## 주요 기능

- 강남, 판교, 시흥 지역 시나리오 지원
- 실제 도로 기반 경로, 남은 거리 및 예상 시간 안내
- 휴대폰 GPS를 이용한 차량 위치·이동 방향 실시간 표시
- 주행 중 GPS 위치를 중심으로 지도 자동 추적
- 지나간 경로는 회색, 남은 경로는 파란색으로 구분
- 주행 시작, 회전, 이탈, 도착 등에 대한 음성 안내
- 구간별 주행·휴식 타이머와 종료 전 경고 제공
- 출발지, 도착지, 경유점 및 주행 경로 직접 수정
- 완료한 주행 기록을 XLSX 또는 CSV 파일로 다운로드
- 수정된 경로와 시나리오 데이터를 JSON으로 백업·복원
- PC, 태블릿, 모바일 세로·가로 화면을 지원하는 반응형 UI

## 휴대폰에서 사용하는 방법

1. 배포 주소를 Chrome 또는 Samsung Internet에서 엽니다.
2. GPS 위치 권한을 허용합니다.
3. 지역을 선택합니다.
4. 시나리오를 선택합니다.
5. 주행할 구간을 확인합니다.
6. `▶ 시작` 버튼을 누릅니다.
7. GPS 이동이 감지되면 지도가 차량 위치를 자동으로 따라갑니다.
8. 구간 완료 후 다음 구간으로 이동합니다.
9. 작업 종료 후 `오늘 기록 다운로드` 버튼으로 주행 기록을 저장합니다.
