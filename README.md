# 자율주행 데이터 취득 네비게이션

차량 기반 자율주행 데이터 취득 작업을 지원하기 위한 모바일·웹 내비게이션 PWA입니다.  
차량과 이름을 선택한 뒤 지역과 수집 시나리오를 고르면, 실제 도로 경로·현재 주행 구간·예상 시간·GPS 위치·턴바이턴 안내를 확인할 수 있습니다.

## 기술 구성

| 구분 | 사용 기술 |
|------|-----------|
| 화면 | HTML5, CSS3, JavaScript |
| 지도 | Leaflet 1.9.4 |
| 지도 타일 | OpenStreetMap |
| 도로 경로 | OSRM Routing API |
| GPS | Browser Geolocation API |
| 음성 안내 | Web Speech API (한국어 TTS) |
| 엑셀 생성 | SheetJS (XLSX) / CSV 폴백 |
| 데이터 저장 | Browser localStorage · sessionStorage |
| 오프라인 지원 | Service Worker |
| 배포 | GitHub Pages |
| 앱 형태 | Progressive Web App (PWA) |

## 주요 기능

### 로그인 및 세션
- 차량(토레스 1–4호차) · 이름 선택 후 입장
- 세션 동안 헤더에 현재 차량·이름 배지 표시

### 지역 및 시나리오
- 강남, 판교, 시흥 지역 시나리오 지원
- 시나리오 직접 수정 모드 (출발·도착 이름, 구간 추가·삭제·순서 변경)
- 구간 목록 드래그 앤 드롭 순서 변경

### 경로 및 지도
- OSRM 기반 실제 도로 경로, 하버사인 공식으로 계산한 정확한 거리
- 지나간 경로는 회색, 남은 경로는 파란색으로 구분
- 출발지·도착지·경유점 직접 수정 (경유점 추가·삭제, 드래그 이동)
- 우측 패널 너비 · 경로 안내 패널 높이 드래그 리사이즈
- 현재 구간을 네이버 지도로 바로 열기

### GPS 및 위치
- 실시간 차량 위치·이동 방향 표시, 주행 중 지도 자동 추적
- GPS 수신 상태 표시 (양호 / 보통 / 불량 / 수신 없음, 색상 + 점멸)
- 도착지 80m 이내 접근 시 다음 구간 자동 이동
- 경로 이탈 감지 (120m 임계값, 실제 경로 좌표 기반)

### 턴바이턴 안내
- 지도 위 오버레이로 다음 회전 방향·거리·도로명 표시 (500 m / 200 m / 50 m 사전 안내)
- 전체 경로 안내 패널 (접기·펼치기 가능)
- Web Speech API 한국어 TTS 음성 안내 (출발, 회전, 이탈, 도착, 타이머 경고, 휴식)
- 음소거 토글 버튼 (헤더 🔊)

### 타이머
- 구간별 주행 타이머 · 휴식 타이머
- 주행 시간 5분 이하 남을 시 빨간색 경고 + 음성 안내

### 주행 기록
- 출발·도착 시각, 구간 ID, 출발지·도착지 기록
- 구간 시작·종료 시점 GPS 좌표 캡처 → Nominatim 역지오코딩 (줌 18 → 16 → 12 → 8 다단계 재시도)
- 오늘 기록 XLSX 다운로드 (SheetJS 미로드 시 CSV 자동 폴백)

### 설정 저장 및 이식
- 경로 수정·시나리오 수정·구간 변경·주행 기록을 JSON 파일로 내보내기·가져오기
- 새 버전 HTML 파일로 교체 후 설정 파일을 가져오면 모든 변경사항 즉시 복원
- 마지막 저장 일시 및 가져온 파일명 헤더에 표시

### UI
- 현재 구간 카드 · 구간 목록 접기·펼치기 (모바일 화면 공간 절약)
- PC · 태블릿 · 모바일 세로·가로 화면 반응형 지원
- 앱 버전 표시 (헤더)

    ├── icon-192.png
    └── icon-512.png

| 파일 | 설명 |

| `icons/` | 홈 화면과 PWA 설치에 사용하는 아이콘 |

## 휴대폰에서 사용하는 방법

1. 배포 주소를 Chrome 또는 Samsung Internet에서 엽니다.
2. GPS 위치 권한을 허용합니다.
3. 차량과 이름을 선택하고 **입장하기**를 누릅니다.
4. 지역과 시나리오를 선택합니다.
5. 주행할 구간을 확인합니다.
6. **▶ 시작** 버튼을 누릅니다.
7. GPS 이동이 감지되면 지도가 차량 위치를 자동으로 따라갑니다.
8. 구간 완료 후 다음 구간으로 이동합니다.
9. 작업 종료 후 **오늘 기록 다운로드** 버튼으로 주행 기록을 저장합니다.

> **💡 새 버전 파일 받을 때:** 먼저 설정 내보내기로 JSON을 저장한 뒤, 새 파일을 열고 설정 가져오기로 복원하세요.
